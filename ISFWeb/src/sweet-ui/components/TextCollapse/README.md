### 这个控件叫什么

文本折叠

### 何时使用

需要省略显示文本，点击展开全部

```jsx
    需要省略显示文本，点击展开全部需要省略显示文本，点击展开全部需要省略显示文本，点击展开全部需要省略显示文本，点击展开全部

```
