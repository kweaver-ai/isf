### 何时使用

设置主题色

### 说明

* 点击色块弹出调色面板，点击弹出面板里的文本框会自动选中文本内容
* `#` 输入框最多输入6位，仅支持数字0-9，A-F(不区分大小写，输入小写确认后自动转换为大写)
* `R／G／B` 输入框最多输入3位，仅支持0-255
* 检测不到输入值对应的颜色时会在文本框失焦后还原为上一次更改的合法值
* 收起面板且选中颜色改变时触发`onValueChange`事件，传递参数为16进制颜色值，格式如'#DD3333'

### 示例

#### 1. 基本使用

```jsx
<ColorPicker value={'6775CD'} onValueChange={(event) => console.log(event.detail)}/>

```

#### 2. 禁用状态

```jsx
<ColorPicker disabled={true}/>

```
